Project Overview

Designed and developed a comprehensive suite of Power BI dashboards to support public health decision-making during the COVID-19 pandemic. The dashboards integrated data from multiple sources to provide actionable insights for healthcare administrators, policymakers, and the general public.

Challenge

During the COVID-19 crisis, decision-makers faced several data visualization challenges:

  • Fragmented data across multiple systems and jurisdictions
  • Difficulty understanding complex pandemic trends from raw numbers
  • Need for real-time updates to support rapid decision-making
  • Varying technical expertise among dashboard users
  • Ensuring data accuracy and consistency across visualizations

Technical Implementation

The dashboard suite was built with a focus on performance, accuracy, and user experience:

  • Data Integration: Power Query for ETL from multiple sources (APIs, databases, flat files)
  • Data Model: Star schema design for optimal performance
  • Visualizations: Custom DAX measures for complex calculations
  • Deployment: Power BI Service for cloud hosting and automatic refresh
  • Security: Row-level security for sensitive data protection
  • Automation: Scheduled data refresh and alert mechanisms

Key Features

The dashboards included several innovative features that enhanced their utility:

  • Drill-through Capabilities: Users could click on summary visualizations to see detailed underlying data
  • What-If Analysis: Interactive parameters allowing users to model different scenarios
  • Mobile Optimization: Responsive design ensuring usability on all devices
  • Export Functionality: One-click export of data and visualizations for reporting
  • Alert System: Automated notifications when key metrics exceeded thresholds

Impact & Results

The dashboard suite transformed how organizations responded to the pandemic:

  • Reduced time to identify emerging hotspots from days to hours
  • Enabled data-driven allocation of limited healthcare resources
  • Improved public understanding of pandemic trends through accessible visualizations
  • Supported vaccination campaign planning with real-time coverage data
  • Facilitated cross-agency collaboration through shared data insights
  • Received recognition from health authorities for clarity and usefulness
